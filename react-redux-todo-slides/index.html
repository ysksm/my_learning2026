05 / Slice の骨組み

state と操作を一緒に定義する

src/features/todo/application/todoSlice.ts
import { createSlice, type PayloadAction } from '@reduxjs/toolkit';
import type { Todo, CreateTodoInput } from '../domain/todo';

type TodoState = { items: Todo[] };
const initialState: TodoState = { items: [] };

const todoSlice = createSlice({
  name: 'todo', initialState,
  reducers: { /* 次のスライドで CRUD を追加 */ },
});
createSlice が作るもの：reducer と action creator。状態・操作名・状態を変える処理を同じファイルに置くのが RTK の基本です。

06 / CRUD reducers

追加・更新・削除を定義する

todoSlice.ts の reducers 内
addTodo: (state, action: PayloadAction<CreateTodoInput>) => {
  state.items.push({ id: crypto.randomUUID(), title: action.payload.title, completed: false });
},
toggleTodo: (state, action: PayloadAction<string>) => {
  const todo = state.items.find(item => item.id === action.payload);
  if (todo) todo.completed = !todo.completed;
},
renameTodo: (state, action: PayloadAction<{ id: string; title: string }>) => {
  const todo = state.items.find(item => item.id === action.payload.id);
  if (todo) todo.title = action.payload.title;
},
deleteTodo: (state, action: PayloadAction<string>) => {
  state.items = state.items.filter(item => item.id !== action.payload);
},
Immer：RTK が安全な不変更新に変換するため、push やプロパティ代入で書けます。更新は「id + 新しい値」、削除は「id」だけを payload に渡します。

08 / Store

slice を一つの store に登録する

src/app/store.ts
import { configureStore } from '@reduxjs/toolkit';
import todoReducer from '../features/todo/application/todoSlice';

export const store = configureStore({
  reducer: { todo: todoReducer },
});

export type RootState = ReturnType<typeof store.getState>;
export type AppDispatch = typeof store.dispatch;
暗記ポイント：reducer: { todo: todoReducer } のキーが、state.todo の名前になります。

10 / Presentation

画面から state を読み、action を送る

src/features/todo/presentation/TodoApp.tsx
const [title, setTitle] = useState('');
const todos = useSelector((state: RootState) => state.todo.items);
const dispatch = useDispatch<AppDispatch>();

<form onSubmit={(e) => {
  e.preventDefault();
  if (title.trim()) { dispatch(addTodo({ title: title.trim() })); setTitle(''); }
}}>
  <input value={title} onChange={e => setTitle(e.target.value)} />
  <button>追加</button>
</form>
{todos.map(todo => <li key={todo.id}>
  <input type="checkbox" checked={todo.completed} onChange={() => dispatch(toggleTodo(todo.id))} />
  {todo.title} <button onClick={() => dispatch(deleteTodo(todo.id))}>削除</button>
</li>)}
流れ：イベント → dispatch(action(payload)) → reducer → store 更新 → useSelector の画面が再描画。
